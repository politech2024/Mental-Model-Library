JSON Schema Reference

Use this format when creating JSON files for import.

Schema .json 
{
  "id": "unique-id",
  "name": "Model Name",
  "thinker": "Author Name",
  "era": "ancient|medieval|earlymodern
         |industrial|digital|aiera",
  "year": "2024",
  "domain": "politics|science|strategy
         |economics|finance|health
         |technology|culture|law
         |business|environment|identity",
  "lens": "power|risk|incentives|feedback
         |ethics|optionality|time
         |asymmetry|coordination",
  "confidence": "high|medium",
  "description": "One-line summary",
  "tags": ["Tag1"],
  "contextMatch": ["high-stakes"],
  "youtubeId": "dQw4w9WgXcQ",
  "whatItIs": "Short explanation",
  "whyExists": "Historical context",
  "coreMechanism": ["Step 1"],
  "clarifies": ["Clarification"],
  "errorsPrevented": ["Error"],
  "whenNotToUse": "Conditions",
  "historicalExample": "Example",
  "relatedModels": ["model-id"]
}

Field Guide

Required

id, name, thinker, era, year, domain, lens, confidence, description

Recommended

tags, contextMatch, whatItIs, whyExists, coreMechanism, relatedModels

Optional

youtubeId, clarifies, errorsPrevented, whenNotToUse, historicalExample

Methodology & Sources

Selection Criteria

Mental models in this library are selected based on three criteria: historical provenance (documented use by decision-makers), empirical support (evidence from behavioral sciences or historical outcomes), and conceptual clarity (clearly defined mechanisms and boundaries of application).

YouTube Videos

Each model's detail page features a curated YouTube video. To set or change a video, open any model and click the red "Set Video" button. Paste any YouTube URL or video ID. Your changes are saved in your browser's local storage.

Confidence Ratings

  • High confidence: Multiple independent historical sources, empirical validation, and consistent application across domains.
  • Medium confidence: Strong theoretical foundation with some historical support; may have limited empirical testing or domain-specific constraints.

Limitations

This library presents frameworks, not prescriptions. Mental models are tools for structuring thought, not algorithms for optimal decisions. Context matters. Historical effectiveness does not guarantee future applicability.

Primary Sources

Models are drawn from primary historical texts, peer-reviewed research in decision science, and documented practices of institutions with long track records of consequential decision-making.
